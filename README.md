> **AI-powered experimentation platform** with Bayesian sequential testing, automated hypothesis generation and LLM-written result narratives. **3x faster decisions**, **< 2% false positive rate**.

## 🏗️ Testing Methods
| Method | When | Advantage |
|--------|------|-----------|
| **Bayesian** | Default | Continuous monitoring, no peeking problem |
| **Sequential (SPRT)** | Need speed | 40% fewer samples needed |
| **Multi-Armed Bandit** | Live traffic | Maximizes reward during test |
| **Causal Inference** | Observational | Handles confounding variables |

## 🤖 AI Features
- **Hypothesis generator**: suggests what to test based on data patterns
- **Sample size calculator**: optimal power analysis per metric type
- **Result narrator**: LLM explains results to non-technical stakeholders
- **Interaction detector**: finds unexpected metric interactions
